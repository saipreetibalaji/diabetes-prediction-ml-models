# Diabetes Prediction using Machine Learning

## Overview

This project focuses on predicting diabetes using various Machine Learning classification models.
It involves data preprocessing, exploratory data analysis, feature importance evaluation, and comparison of multiple models based on performance metrics.

---

## Objectives

* Predict diabetes based on health-related features
* Compare performance of different ML models
* Identify the most effective model for classification
* Analyze feature importance influencing predictions

---

## Methodology

### Data Preprocessing

* Data cleaning and inspection
* Handling missing values
* Feature scaling
* Train-test split

---

### Exploratory Data Analysis

* Distribution plots
* Feature relationships
* Correlation analysis

---

### Models Used

* Random Forest
* Gradient Boosting
* XGBoost
* CatBoost
* LightGBM
* Decision Tree
* K-Nearest Neighbors (KNN)

---

## Results

* **Random Forest achieved highest accuracy: 83%**
* LightGBM: 81.5%
* CatBoost: 80.5%
* Other models showed competitive performance

---

* Accuracy

* Confusion Matrix
* ROC Curve
